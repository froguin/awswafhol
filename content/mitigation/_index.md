- AWS WAF 를 이용하여 웹 액세스 제어 목록(웹 ACL)을 생성하고 필요한 Rule을 추가하는 방법을 살펴보도록 하겠습니다. 
- AWS Manaed Rule을 이용해서 이전 실습에서 진행하였던 SQL Injection 공격과 XSS 공격을 차단해 봅니다. 
- custom Rule을 이용해서 특정 국가 및 특정 user-agent 에서 유입되는 트래픽을 차단해 봅니다.
